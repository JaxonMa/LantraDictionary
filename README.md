# Lantra Dictionary 蓝雀词典
## 项目介绍
这是一个在线词典，使用 flask 开发，可以查询汉字的读音、部首、详细解释等信息，同时提供查词服务，可以帮助用户快速了解一个词语的释义。

## 注意事项
本项目的前端代码为 Vibe Coding 生成，仅Python代码是由人工编写的。该项目是对 Vibe Coding 的一次尝试，尝试重构已有的项目能否达到更好的效果。
事实证明通过 Vibe Coding，开发效率及编码体验都得到了很大的提升，并且达到了预期的效果。

## 开发环境
> Python 3.12

使用了以下Python包：
> flask == 3.1.2
> flask-cors == 6.0.2
> ijson == 3.4.0
> langid == 1.1.6
